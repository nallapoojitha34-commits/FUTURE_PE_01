# AI SEO Blog & Content Cluster Generator

## Project Overview

This project demonstrates how Prompt Engineering can be used to generate SEO-optimized blog content for local businesses.

## Business Chosen

FreshMart Supermarket

## Objective

To create:
- SEO-friendly blog content
- Content cluster ideas
- Structured prompts for AI content generation

## Files Included

### prompts.md
Contains the prompt used to generate SEO blog content.

### blog.md
Contains the AI-generated SEO blog article and related blog topics.

## Tools Used

- ChatGPT
- GitHub

## Outcome

Successfully generated an SEO-friendly blog article and content cluster ideas for a local supermarket business.
