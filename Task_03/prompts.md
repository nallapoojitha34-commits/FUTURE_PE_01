# SEO Blog Generator Prompt

Create an SEO-optimized blog post for a local supermarket named FreshMart Supermarket.

Requirements:
- Create a catchy SEO-friendly title
- Write an introduction
- Include H1, H2, and H3 headings
- Explain the benefits of buying fresh vegetables from local supermarkets
- Add practical tips for customers
- Write a conclusion
- Keep the content easy to read and informative
- Use keywords naturally throughout the article

Also suggest 5 related blog topics for content clustering.
