# AI Content Marketing using UGC Ads

## Project Overview
This project focuses on creating AI-generated UGC (User Generated Content) advertisement content for a skincare product called GlowFresh Face Wash.

## Product Chosen
GlowFresh Face Wash

## Objective
To generate engaging UGC ad scripts, hooks, CTAs, and captions that can be used for Instagram Reels, Facebook Ads, and YouTube Shorts.

## Tools Used
- ChatGPT
- GitHub
- Google Docs

## Deliverables
- UGC Ad Scripts
- Hooks
- Call-to-Action (CTA) Content
- Social Media Captions
- Structured Prompts
